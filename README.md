This repository contains the proof-of-concept implementation for the Improved Random Forest (I-RF) component of our hybrid architecture. Due to strict institutional Non-Disclosure Agreements (NDA) and child privacy regulations, the original administrative dataset cannot be publicized. Instead, we provide a synthetic dummy dataset (generate_synthetic_data.py) that mirrors the extreme class imbalance (1:20 recidivism ratio) discussed in the study. The training script (train_irf_model.py) demonstrates the integration of cost-sensitive learning and Tree-structured Parzen Estimator (TPE) Bayesian optimization.
